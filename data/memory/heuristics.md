# Learned Heuristics

This file is populated automatically by the evaluation step.
Claude reads this file before making predictions to apply learned patterns.

## Format

Each heuristic entry should include:
- **Pattern**: What was observed
- **Adjustment**: What to do about it
- **Confidence**: How reliable this pattern is (based on sample size)
- **Added**: Date when discovered
- **Source**: Which gameweek evaluation produced this
